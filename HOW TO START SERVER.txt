HOW TO START THE LOCAL SERVER (Next.js app)

FIRST TIME ONLY:
1. Open Cursor with this project folder.
2. Open the terminal: press Ctrl+` (backtick, key above Tab).
3. Run: npm install

EVERY TIME YOU WORK:
1. Open the terminal: press Ctrl+`.
2. Run: npm.cmd run dev
   (If npm works normally in your shell, npm run dev is fine too.)
3. Open in browser: http://localhost:3000
4. To stop the server: click in terminal, then press Ctrl+C

IF POWERSHELL SHOWS "RUNNING SCRIPTS IS DISABLED":
- Quick fix (recommended): use npm.cmd run dev
- Optional permanent fix:
  Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
  Then restart terminal and run npm run dev

IMPORTANT:
- This project now uses Next.js.
- Old static app files (index.html, script.js, styles.css, start-server.bat) are legacy and are not used by the new app server.
